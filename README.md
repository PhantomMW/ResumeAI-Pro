# 🚀 ResumeAI Pro

![Python](https://img.shields.io/badge/Python-3.10-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.58-FF4B4B?logo=streamlit)
![Google Gemini](https://img.shields.io/badge/Google-Gemini-blue?logo=google)

An AI-powered Resume Analyzer & Career Assistant built with **Streamlit** and **Google Gemini AI**.

---

## ✨ Features

- 📄 Resume Analysis
- 🎯 ATS Compatibility Score
- 📊 Resume Quality Score
- 🤝 Job Match Analysis
- 🧠 Skills Detection
- ⚠️ Missing Skills Analysis
- 💌 Cover Letter Generator
- ✨ Resume Rewriter
- 💼 LinkedIn Summary Generator
- 🛣 AI Career Advisor
- 🤖 Resume Chat Assistant
- 📑 Professional PDF Report

---

## 🛠 Technologies

- Python
- Streamlit
- Google Gemini API
- ReportLab
- Plotly
- PDFPlumber
- Python-docx

---

## ⚙️ Installation

```bash
git clone https://github.com/0115455/ResumeAI-Pro.git

cd ResumeAI-Pro

pip install -r requirements.txt

streamlit run app.py
```
